18.7. Инсталляция операционной системы на виртуальную машину

Инсталляция операционной системы на вновь созданную виртуальную машину осуществляется точно так же, как инсталляция ОС на физический компьютер:
  1. Установите в дисковод CD-ROM или гибкий диск с установочными файлами операционной системы.

  2. Включите виртуальную машину -- щелкните мышкой по клавише "Power On" на "корпусе" виртуальной машины.

  3. Следуйте инструкциям, которые выдает программа инсталляции.

На сайте фирмы VMWare можно найти более подробные рекомендации по установке на ВМ конкретных операционных систем:

  • Windows NT 4.0
  • Windows 95
  • Windows 98
  • Windows 2000
  • DOS / Windows 3.1
  • Caldera OpenLinux
  • Corel Linux OS
  • Red Hat Linux
  • SuSE Linux
  • FreeBSD
  • Novell NetWare

12.3. Установка FreeBSD по FTP с FTP Сервера

12.3.1. Меню Выбора Носителя/Метода Установки

Рисунок 12-3. Выбор FTP Сервера

Выбирите "Install from an FTP server", затем [OK] и нажмите ENTER.

12.3.2. Выбор FTP Сайта-Site из Menu

Рисунок 12-4. Выбор сайта FTP из Меню

Выбирите ближайший к вам ftp сайт содержащий нужный вам релиз дистрибутива FreeBSD для достижения максимальной скорости его скачивания. Основной сайт заведомо содержит все необходимые версии дистрибутивов, вы в свою очередь можете воспользоваться его зеркалами, предварительно проверив содержат ли они требуемый вам дистрибутив.

Выбирите предпочтительный для вас ftp сайт, выбирите [OK] и нажмите ENTER.

12.3.3. Задание FTP сайта отсутствующего в списке Меню

Рисунок 12-5. Задание FTP сайта отсутствующего в Меню

Если вы хотите указать адрес сайта отсутствующего в списке, выбирите пункт "Specify some other ftp site by URL", затем выбирите [OK] и нажмите ENTER.

Рисунок 12-6. Ввод URL адреса FTP сайта

На экране появится строка в которой необходимо указать адрес, подобно тому как это сделано в примере. По завершении ввода адреса, выбирите [OK] и нажмите ENTER.

12.3.4. Подсоединение к FTP Сайту через PPP для установки FreeBSD

Рисунок 12-7. Выбор PPP из Меню Сетевого Интерфейса

На следующем этапе укажите соответствующие опции подключения к сети для соединения с удаленным ftp сайтом:

В данном примере, внешний модем подключен к порту COM2. Соответственно выбирается опция "PPP interface on device /dev/cuaa1 (COM 2)". После чего указываем [OK] и нажимаем ENTER для продолжения.

Рисунок 12-8. Настройка IPV6 Протокола

Для нашей локальной сети вполне достаточно текущего Internet протокола (IPv4), поэтому выбираем [No] и нажимаем [ENTER].

Если же вы хотите попробовать новый протокол (IPv6), выбирите [Yes] и нажмите [ENTER]. После чего необходимо подождать некоторое время пока не завершиться поиск RA серверов.

Рисунок 12-9. Настройка DHCP

В данном примере отсутствует DHCP (Dynamic Host Configuration Protocol), поэтому выбираем [No] и нажимаем [ENTER].

Если будет выбрана опция [Yes], запустится dhclient и если он найдет сервер раздающий свободные адреса, то автоматически заполнит таблицу сетевой конфигурации. В качестве справочной информации воспользуйтесь The FreeBSD Handbook Handbook руководством.

Рисунок 12-10. Настройки для интерфейса ppp0

Заполните требуемые поля необходимой информацией, для перехода по полям используйте клавишу TAB. Как минимум, вам необходимо указать ip-address сервера имен(DNS). Заполнение данных полей будет зависеть от требований и настроек сервера PPP с которым будет произведено соединение.

После ввода требуемой информации, используя TAB выбирите [OK] и для завершения нажмите ENTER.

Рисунок 12-11. Установка DTE скорости работы COM порта с Модемом

Нажмите ENTER для продолжения.

Рисунок 12-12. Задание IP адреса Провайдера

Нажмите ENTER для продолжения.

Рисунок 12-13. Использование PAP или CHAP

Если данный метод используется, выбирите [Yes] и нажмите ENTER для продолжения.

Рисунок 12-14. Задание UserName для PPP Соединения

Введите имя пользователя которое должно быть использовано для соединени, выбирите [OK] и нажмите ENTER для продолжения.

Рисунок 12-15. Задание Password для PPP Соединения

Введите пароль для указанного ранее имени пользователя, выбирите [OK] и нажмите ENTER для продолжения

Рисунок 12-16. Задание Номера Телефона для PPP Соединения

Введите телефонный номер вашего провайдера, выбирите [OK] и нажмите ENTER для продолжения.

Рисунок 12-17. Использование Тонального метода дозвона

Если у вас поддерживается тональный дозвон, нажмите ENTER для продолжения.

Рисунок 12-18. Сообщения после запуска PPP

Нажмите ALT-F3 чтобы перейти на третий виртуальный экран для просмотра сообщений протокола PPP (Point to Point Protocol):

Рисунок 12-19. Соединение с Провайдером

Введите команду dial в строке приглашения и если в итоге вы получите три заглавных буквы P (PPP), значит соединение успешно установлено. После этого нажмите ALT-F1 чтобы вернуться на экран процедуры установки.

Рисунок 12-20. Продолжение Установки

Нажмите ENTER для продолжения.

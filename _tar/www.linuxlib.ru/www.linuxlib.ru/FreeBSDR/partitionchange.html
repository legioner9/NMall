9.4. Создание Партиций-Partitions с Собственными Параметрами

Сохраните расчеты авто-конфигуратора для всех фаловых систем (/,swap, /var, /usr). Затем, используя клавиши стрелок, выбирите первую партицию ( / ) и удалите ее нажатием D. Аналогично удалите все остальные партиции и ваш экран будет выглядеть подобно нижнему с небольшими различиями, такими как значение в поле "Free:".

Рисунок 9-4. Sysinstall Disklabel Редактор

Теперь можно самостоятельно создать партиции, в том же порядке что они были созданы авто-конфигуратором.

9.4.1. Создание Root Партиции "/"

Для создания root партиции, "/", сначала нажмите клавишу C и появится диалоговое окно в котором будет указан размер в блоках всего доступного свободного дискового пространства:

Рисунок 9-5. Свободное место для Root Партиции

Чтобы установить размер "/" партиции равный 100M, сначала нужно с помощью клавиши BACKSPACE удалить предложенное значение и ввести требуемое - 100M:

Рисунок 9-6. Изменение размера Root Партиции

Выбираем [OK] и нажимаем ENTER. После чего появится следующее диалоговое окно:

Рисунок 9-7. Выбор типа Root Партиции

Root Партиция должна быть файловой системой. Для этого выбираем "FS", затем [OK] и нажимаем ENTER.

В следующем диалоговом окне, необходимо будет задать точку монтирования root партиции - директорию куда она будет смонтирована. Набираем "/", выбираем [OK] и нажимаем ENTER.

Рисунок 9-8. Выбор точки монтирования Root Партиции

9.4.2. Создание "swap" Партиции

Для создания партиции нажмите клавишу C, появится диалоговое окно в котором будет указан размер в блоках всего доступного свободного пространства:

Рисунок 9-9. Выделение свободного места под Swap

Введите нужный вам размер для "SWAP" партиции. Обычно, значение swap расчитывают как количество RAM памяти умноженное на два. В данном примере, значение "SWAP" партиции установлено в 516M. Сначала необходимо клавишей BACKSPACE удалить предлагаемый размер и затем задать свой:

Рисунок 9-10. Редактирование размера Swap

Затем выбрать [OK] и нажать ENTER. После чего появится диалоговое окно выбора Типа партиции, для swap необходимо выбрать SWAP, затем [OK] и нажать ENTER. В отличие от FS, у нас не появится диалоговое окно для указания точки монтирования, поскольку для SWAP она не нужна

Рисунок 9-11. Выбор типа Партиции Swap

9.4.3. Создание "/var" Партиции

Для создания "/var" партиции, нажмите [C], появится следующее диалоговое окно:

Рисунок 9-12. Выделение свободного места для /var Партиции

Введите требуемый размер. В указанном примере, размер "/var" установлен в 100M. Сперва удаляем заданный размер с помощью [BACKSPACE] и затем вводим требуемый:

Рисунок 9-13. Редактирование размера /var Партиции

Затем выбираем [OK] и нажимаем [ENTER]. После чего появится следующее диалоговое окно:

Рисунок 9-14. Выбор типа Партиции для /var

Выбираем нужный тип партиции FS, затем выбираем [OK] и нажимаем [ENTER], а в следущем меню, в качестве точки монтирования задаем директорию "/var"

Рисунок 9-15. Выбор точки монтирования /var

9.4.4. Создание "/usr" Партиции

Для создания "/usr" партиции, нажмите [C], появится диалоговое окно:

Рисунок 9-16. Выделение свободного места для /usr Партиции

В данном примере отводим все оставшееся место под последнюю нашу партицию "/usr", выбираем [OK] и нажимаем [ENTER]. После чего появится следующее диалоговое окно:

Рисунок 9-17. Выбор типа Партиции для /usr

в котором необходимо задать в качестве точки монтирования имя директории. Набираем "/usr" для партиции usr, выбираем [OK] и нажимаем [ENTER]

Рисунок 9-18. Выбор точки монтирования /usr

После указанных операций окно FreeBSD DiskLabel Редактора будет выглядеть подобно нижнему, отличие будет лишь в размерах. Диск размечен, для завершения необходимо нажать [Q].

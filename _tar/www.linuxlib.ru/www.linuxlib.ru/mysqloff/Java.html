8.6 Взаимодействие MySQL и Java (JDBC)

Имеется два поддерживаемых драйвера JDBC для MySQL (драйвер mm и драйвер Reisin JDBC). Копию драйвера mm можно найти на http://mmmysql.sourceforge.net/ или на http://www.mysql.com/Downloads/Contrib/, а драйвера Reisin - на http://www.caucho.com/projects/jdbc-mysql/index.xtp. По вопросам, касающимся документации, обращайтесь к любой документации по JDBC, а по вопросам, касающимся присущих MySQL специфических особенностей, - к собственной документации по конкретному драйверу.

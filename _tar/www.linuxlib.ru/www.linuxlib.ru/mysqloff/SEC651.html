B.2 Клиенты

  • Графические клиенты
    • http://www.ideit.com/products/dbvis/ DbVisualizer, свободнораспространяемый клиент JDBC для графической визуализации данных и структуры нескольких баз данных одновременно. Разработка Innovative-IT Development AB.
    • http://www.mysql.com/downloads/gui-clients.html MySQLGUI, веб-страница, посвященная клиенту MySQL GUI. Автор Синиша (Sinisa) из MySQL AB.
    • http://www.mysql.com/Downloads/Contrib/mysql_navigator_0.9.0.tar.gz MySQL Navigator представляет собой клиентскую программу графического пользовательского интерфейса (GUI) для сервера баз данных MySQL, распространяемую под лицензией GPL. Назначение MySQL Navigator состоит в обеспечении удобного клиентского интерфейса для серверов баз данных MySQL при поддержке многих операционных систем и языков. В настоящее время можно производить импорт/экспорт базы данных, вводить запросы, получать результирующие наборы, редактировать и запускать скрипты, добавлять, изменять и удалять пользователей и получать информацию о клиенте и сервере. Использует QT 2.2. Веб-страница для MySQL Navigator расположена на http://sql.kldp.org/mysql/.
    • http://www.mysql.com/Downloads/Win32/secman.zip GUI для управления пользователями и безопасностью для MySQL под Windows. Автор Мартин Джеремик (Martin Jeremic). Домашняя страница GUI безопасности для MySQL находится на http://jsoft.webjump.com/.
    • http://www.mysql.com/Downloads/Contrib/kmysqladmin-0.4.1.tar.gz.
    • http://www.mysql.com/Downloads/Contrib/kmysqladmin-0.4.1-1.src.rpm.
    • http://www.mysql.com/Downloads/Contrib/kmysqladmin-0.4.1-1.i386.rpm Инструмент администрирования сервера MySQL с использованием QT/KDE. Протестировано только под Linux.
    • http://www.mysql.com/Downloads/Contrib/mysql-admin-using-java+swing.tar.gz Java-клиент с использованием Swing, автор Фреди Фишер (Fredy Fischer [email protected]). Последнюю версию всегда можно найти на http://www.trash.net/~ffischer/admin/.
    • http://www.mysql.com/Downloads/Win32/MySQL-Maker-1.0.zip. Условно-бесплатная версия клиента MySQL для Windows. Представляет собой редактор типа WYSIWYG, позволяющий создавать, модифицировать и удалять базы данных и таблицы. Можно изменить структуру полей и добавить, изменить и удалить данные в этих таблицах напрямую без драйвера ODBC. Домашняя страница программы MySQL Maker расположена на http://www.presult.de/presult/frames/fs_mysqlmaker.html.
    • http://www.mysql.com/Downloads/Contrib/mysqlwinadmn.zip GUI для Windows (только бинарный код) для администрирования базы данных, автор Дэвид Б. Мензел (David B. Mansel, [email protected]).
    • http://home.online.no/~runeberg/myqa/ MyQA представляет собой базированный на Linux клиент запросов для сервера баз данных MySQL. MyQA позволяет вводить SQL-запросы, выполнять их и просматривать результаты, и все это при помощи графического пользовательского интерфейса. Данный GUI приблизительно похож на GUI клиента 'Query Analyzer', который имеется в сервере MS SQL.
    • http://www.opex.atnet.ru/mysqlmanager/ MySQL Manager представляет собой написанную на Java графическую управляющую программу для сервера MySQL.
    • http://www.mysql.com/Downloads/Win32/netadmin.zip Инструмент администрирования для MySQL под Windows 95/98 и Windows NT 4.0. Протестировано только с версиями MySQL 3.23.5 - 3.23.7. Написано с использованием компонентов Tmysql. Можно писать запросы, показывать таблицы, индексы, табличную структуру, контролировать пользователя, хост, базу данных и так далее. Данная программа является бета-версией и еще имеет ряд ошибок. Вы можете протестировать эту программу во всех ее возможностях. Пожалуйста, присылайте ошибки и советы Марко Суес (Marco Suess [email protected]). Исходный URL http://www.it-netservice.de/pages/software/.
    • http://www.mysql.com/Downloads/Win32/netadmin2.zip Новая версия программы netadmin. Более подробно смотрите выше.
    • http://www.mysql.com/Downloads/Win32/ARTADMIN203.EXE Клиент MySQL разработки Atronic дляWindows 2.0.3.0. Домашнюю страницу разработчика можно найти по адресу http://www.artronic.hr/.
    • http://www.mysql.com/Downloads/Win32/W9xstop.zip Служебная программа от Artronic для останова MySQL под win9x.
    • http://bardo.hyperlink.cz/mysqlmon/ Облегченный клиент GUI для Windows.
    • http://www.mysqlfront.de/ MySQLfront является очень хорошим клиентом для Windows со множеством полезных свойств. Автор Энгсар Беккер (Angsar Becker).
    • http://www.dbtools.com.br/ Dbtools, инструмент для управления базами данных MySQL. В настоящее время только для Windows. Некоторые возможности:
      • Управление серверами, базами данных, таблицами, столбцами, индексами и пользователями.
      • "Мастер" импорта структуры и данных из баз данных MS Access, MS Excel, Dbase, FoxPro, Paradox и ODBC.
      • http://www.mysql.com/Downloads/Contrib/KMYENG113.zip GUI администратора для MySQL. Работает только под Windows, без исходного кода. Доступен на английском и японском языках. Автор Митунобу Канеко (Mitunobu Kaneko). Домашняя страница: http://sql.jnts.ne.jp/
    • http://www.mysql.com/Downloads/Contrib/xmysqladmin-1.0.tar.gz X-интерфейс для процессора баз данных MySQL. Позволяет выполнять операции перезагрузки, контроля состояния, контроля функционирования, myisamchk, наделения привилегиями и их отмены, создания баз данных, удаления баз данных, создания, изменения, просмотра и удаления таблиц. Первоначально разработан Джильбертом Терьеном (Gilbert Therrien, [email protected]), но сейчас является всеобщим достоянием и поддерживается MySQL AB.
    • http://www.mysql.com/Downloads/Contrib/xmysql-1.9.tar.gz. Программа Xmysql представляет собой интерфейс для процессора баз данных MySQL. Пригодна для поддержки как простых запросов и таблиц, так и для пакетов запросов. Автор Рик Мехалик (Rick Mehalick, [email protected]). Посвященная xmysql веб-страница находится на http://web.wt.net/~dblhack/. Для работы требует http://bragg.phys.uwm.edu/xforms/ (xforms 0.88).
    • http://www.tamos.net/sw/dbMetrix/ dbMatrix представляет собой распространяемый как Open Source клиент для исследуемых баз данных и выполняемых запросов SQL. Поддерживает MySQL, Oracle, PostgreSQL и mSQL.
    • http://www.multimania.com/bbrox/GtkSQL/ GtkSQL представляет собой инструмент запроса для MySQL и PostgreSQL.
    • http://dbman.linux.cz/ dbMan является написанной на Perl инструментальной программой работы с запросами. Использует DBI и Tk.
    • http://www.mysql.com/Downloads/Win32/Msc201.EXE (Mascon 202)
    • http://www.mysql.com/Downloads/Win32/FrMsc202.EXE (Free Mascon 202) Программа Mascon является мощным GUI для Win32 для администрирования сервера баз данных MySQL. Возможности программы Mascon включают в себя визуальное проектирование таблиц, подключения ко многим серверам, редактирование таблиц с простыми данными и объектами BLOB, установку безопасности, цветовое кодирование в SQL, выполнение дампов и многое другое. Домашняя страница Mascon находится на http://www.scibit.com/Products/Software/Utils/Mascon.asp.
    • http://www.virtualbeer.net/dbui/ DBUI представляет собой графический редактор Gtk баз данных.
    • http://www.rtlabs.com/ MacSQL представляет собой GUI для баз данных MySQL, ODBC и JDBC для операционных систем Mac.
    • http://www.caleb.com.au/ JRetriever является сервисной программой, представляющей собой общий интерфейс для JDBC-совместимых баз данных, написанной с помощью Java 2. JRetriever отображает таблицы/представления базы данных в интерфейсе, похожем на проводник Windows. Пользователи могут извлекать данные, отмечая указателем мыши папку данной таблицы или составляя свои собственные SQL-команды с помощью встроенного SQL-редактора. Данный инструмент был протестирован с Oracle 8 и MySQL как серверными базами данных. Из программных средств поддержки Java эта программа требует JDK 1.3.
    • http://www.jetools.com/products/databrowser/ Программа DataBrowser представляет собой независимый от базы данных кросс-платформенный инструмент доступа к данным. Эта программа более дружественна для пользователя, чем такие инструменты, как SQL Plus, psql (инструменты на базе командной строки). А также более гибкая, чем TOAD, ISQL, PGAccess, интерфейсы которых ограничены какой-либо одной платформой или базой данных.
    • http://www.intrex.net/amit/software/ SQL Console представляет собой автономное java-приложение, позволяющее подключаться к системе баз данных SQL, осуществлять SQL-запросы и обновления. Данная программа имеет простой для применения графический интерфейс пользователя. SQL Console использует JDBC для подключения к системе баз данных и, следовательно, с помощью соответствующего JDBC-драйвера можно использовать эту инструментальную программу для подключения к некоторым наиболее популярным системам баз данных.
    • http://www.mysql.com/Downloads/Contrib/mysql_mmc.zip MySQL MMC представляет собой инструмент управления GUI, разработанный с использованием kdevelop с очень хорошим интерфейсом, полностью сходным с Microsoft Enterprise Tool (для SQL Server) или Sybase Central. Мы можем использовать эту программу для управления сервером, таблицей, индексом, пользователем, а также для редактирования табличных данных в сетке или выполнения SQL-команд посредством Query Analysis.
  • Веб-клиенты
    • http://www.mysql.com/Downloads/Contrib/mysqladmin-atif-1.0.tar.gz Веб-администратор MySQL для таблиц user, db и host. Автор Тим Сэйлер (Tim Sailer), модифицировал Атиф Гафар (Atif Ghaffar [email protected]).
    • http://www.mysql.com/Downloads/Contrib/mysql-webadmin-1.0a8-rz.tar.gz Инструментальная программа, написанная на PHP-FI, для удаленного администрирования баз данных MySQL через Интернет с использованием веб-браузера. Автор Питер Куппелвесер (Peter Kuppelwieser, [email protected]). Обновление Вим Бонис (Wim Bonis, [email protected]). Уже больше не поддерживается!
    • http://www.mysql.com/Downloads/Contrib/mysqladm.tar.gz Написанный на Perl администратор веб-базированной базы данных MySQL. Автор Тим Сэйлер (Tim Sailer).
    • http://www.mysql.com/Downloads/Contrib/mysqladm-2.tar.gz Обновленная версия `mysqladm.tar.gz', автор Хай Тайд (High Tide).
    • http://www.mysql.com/Downloads/Contrib/billowmysql.zip Обновленная версия `mysqladm.tar.gz', автор Янг Гао (Ying Gao). Новейшую версию можно получить с http://civeng.com/sqldemo/ (домашний сайт).
    • http://www.mysql.com/Downloads/Contrib/myadmin-0.4.tar.gz MyAdmin представляет собой веб-базированный администратор MySQL, разработанный Майком Мачадо (Mike Machado). Веб-страница MyAdmin находится на http://myadmin.cheapnet.net/
    • http://www.mysql.com/Downloads/Contrib/phpMyAdmin_2.2.0.tar.gz Набор PHP3-скриптов для администрирования MySQL через Интернет.
    • http://www.phpwizard.net/projects/phpMyAdmin/ phpMyAdmin представляет собой инструментальную программу на PHP3 в духе mysql-webadmin, автор Тобиас Рэтшиллер (Tobias Ratschiller, [email protected]).
    • http://www.mysql.com/Downloads/Contrib/useradm.tar.gz Администратор MySQL на PHP. Автор Офни Томас (Ofni Thomas [email protected]).
    • http://gossamer-threads.com/perl/mysqlman/mysql.cgi Программа MySQLMan имеет функциональность, подобную phpmyadmin, но написана на Perl с использованием HTML-темплейтов. Автор Алекс Крон (Alex Krohn).
  • http://www.mysql.com/Downloads/Contrib/mysql-editor.tar.gz Эти cgi-скрипты на Perl дают возможность редактировать содержание базы данных MySQL. Автор Томас Земан (Tomas Zeman).
  • http://worldcommunity.com/opensource/futuresql/ FutureSQL, разработчик Петер Ф. Браун (Peter F. Brown), это написанное на Perl и использующее MySQL быстродействующее свободно распространяемое приложение, представляющее собой инструментальное средство разработки администрирования расположенной в Сети базы данных. Использует DBI:DBD и CGI.pm. Программа FutureSQL позволяет легко устанавливать конфигурационные файлы для просмотра, редактирования, удаления или же обработки записей из базы данных MySQL. Использует словарь данных, конфигурационные файлы и темплаты и позволяет проводить предварительную и послеоперационную обработку как на полях и записях, так и на операциях.

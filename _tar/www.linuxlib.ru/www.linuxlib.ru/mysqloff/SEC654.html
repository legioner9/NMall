B.5 Средства аутентификации

  • http://www.mysql.com/Downloads/Contrib/ascend-radius-mysql-0.7.2.patch.gz Это патч аутентификации и регистрации с использованием MySQL для Ascend-Radius. Разработка [email protected].
  • http://www.mysql.com/Downloads/Contrib/icradius-0.10.tar.gz (icradius 0.10) http://www.mysql.com/Downloads/Contrib/icradius.README (iceradius readme)
  • http://www.mysql.com/Downloads/Contrib/checkpassword-0.81-mysql-0.6.6.patch.gz Патч аутентификации к MySQL для QMAIL и проверки пароля. Это полезно для управления пользователем (почта, исходящий аккаунт) с помощью MySQL. Разработка [email protected].
  • http://www.mysql.com/Downloads/Contrib/jradius-diff.gz Поддержка MySQL для Radius 2.01 выпуска Livingston. Аутентификация и ведение аккаунта. Автор Жозе де Леон (Jose de Leon, [email protected]).
  • http://www.mysql.com/Downloads/Contrib/mod_auth_mysql-2.20.tar.gz Модуль аутентификации Apache для MySQL. Автор Зив Сураски (Zeev Suraski, [email protected]).
  • http://www.mysql.com/Downloads/Contrib/mypasswd-2.0.tar.gz Дополнение для mod_auth_mysql. Этот небольшой инструмент позволяет добавить/изменить записи пользователя хранимой группы и/или входные пароли в таблицах MySQL. Автор Гарри Брукнер (Harry Brueckner, [email protected]).
  • http://www.mysql.com/Downloads/Contrib/mysql-passwd.README
  • http://www.mysql.com/Downloads/Contrib/mysql-passwd-1.2.tar.gz Дополнение для mod_auth_mysql. Представляет собой двухмодульную систему для использования совместно с mod_auth_mysql.
  • http://www.mysql.com/Downloads/Contrib/pam_mysql.tar.gz Данный модуль осуществляет аутентификацию пользователя в pam, используя MySQL.
  • http://www.mysql.com/Downloads/Contrib/nsapi_auth_mysql.tar Функции программного интерфейса приложений веб-сервера Netscape (NSAPI) для базовой аутентификации пользователей в отношении таблиц MySQL. Автор Юан Джон Джианг (Yuan John Jiang).
  • http://www.mysql.com/Downloads/Contrib/qmail-1.03-mysql-0.6.6.patch.gz Патч для qmail для аутентификации пользователей из таблицы MySQL. Разработка [email protected].
  • http://www.mysql.com/Downloads/Contrib/proftpd-1.2.0rc2-fix-mysql.patch Патч для proftpd1.2.0rc2. Разработка [email protected].
  • http://www.mysql.com/Downloads/Contrib/pwcheck_mysql-0.1.tar.gz Модуль аутентификации для сервера Cyrus IMAP. Автор Арон Ньюсоум (Aaron Newsome).

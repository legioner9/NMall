Тема: [8] Сетевые ресурсы и списки поставщиков
(1998/04/06)

Некоторые из сайтов содержат как техническую информацию, так и осуществляют продажу продуктов; они перечислены дважды.

Библиография CD-Info на http://www.cd-info.com/CDIC/Bibliography.html обновляется чаще, и, вероятно, точнее данного раздела. Там также имеются ссылки на книги, журналы, смежные области в коммерческих онлайновых службах, и прочие полезные вещи.

Тема: [8-1] Информационные ресурсы
(2000/04/03)

Несколько полезных веб страниц. Не забывайте о телеконференциях, перечисленных в разделе (0-5).

http://resource.simplenet.com/
Коллекция файлов и ссылок на URL, связанных с CD-R Mike Richter'а.

http://www.cdpage.com/
Много полезных вещей, включая библиотеку технических статей.

http://www.cd-info.com/
Информация о CD и CD-R, в особенности по технологии и производству.

http://www.osta.org/
Optical Storage Technology Association (OSTA); см. CD-R Q&A .

http://www.sigcat.org/
Домашняя страница SIGCAT (Special Interest Group on CD Applications and Technology).

http://www.adaptec.com/cdrec/
Множество полезной информации по CD-R.

http://www.adaptec.com/support/cdrlist/policies.html
Отрывки из списков рассылки Adaptec CD-R и инструкции по подписке на них.

http://www.westnet.com/~gsmith/cdrecord.htm
Практические советы и опыт работы с CD-R, включая длинную дискуссию по переводу аудиозаписей с других носителей.

http://www.cinram.com/techlibrary/main.html
Техническая библиотека Cinram (ранее Disc Manufacturing, Inc.) Раньше содержала больше информации. Осталось несколько статей.

http://www.cdarchive.com/
Множество информации по CD и CD-R.

ftp://ftp.cdrom.com/pub/cdrom/faq
Информация и FAQ’и телеконференции alt.cd-rom.

http://www.araltd.com/whatscdr.html
Информация по продуктам, а также модная графика.

http://www.hsv.tis.net/disc/
Техническая информация.

http://emf.net/~mal/cdplus.html
Информация по CD Plus (a/k/a CD Extra). [ больше нет по этому адресу? ]

http://www.tc.umn.edu/~erick205/Papers/paper.html
Очень технический доклад по CD.

http://www.ee.washington.edu/conselec/CE/kuhn/cdaudio/95x6.htm
Техническая дискуссия по конструкции CD и трекингу лазера.

http://www.howstuffworks.com/cd.htm
Очень милое объяснение того, как работает CD.

http://www.compulink.co.uk/~xi/hp4020i.htm
Сайт, посвященный HP4020i.

http://www.epool.com/mtc/mtc_02.html
Информация по CD-R вообще, и по носителям Mitsui Toatsu в частности.

http://www.octave.com/library.html
Набор хороших вещей.

http://www.tardis.ed.ac.uk/~psyche/cdda/
Информация и ссылки по извлечению цифрового аудио.

http://music.digidesign.com/html/faqs/faqmlcd.html
FAQ по MasterList CD. Есть интересные заметки.

http://www.geocities.com/SiliconValley/Bay/6665/jdlnet_jvc_cd-r.html
Коллекция информации и полезных советов для новых пользователей вообще, и для пользователей JVC XR-W2010 в частности.

http://www.geocities.com/SiliconValley/Bay/6893/
Информация по рекордерам CDD2600 и CDD3610.

http://www.orangeforum.or.jp/english/
Веб страница "Orange Forum". (CD-R - "Оранжевая книга".) Очень художественно.

http://www.digido.com/
Несколько статей по разным предметам, связанным с аудиозаписью студийного качества.

http://www.hrrc.org/
Коалиция прав домашней звукозаписи (некоторые юридические штучки).

http://members.xoom.com/cdburning/
Рай для CD-R.

http://weber.u.washington.edu/~jrao/maccdr/
CD-R на Macintosh.

http://www.mscience.com/
Веб сайт Media Sciences; хорошая техническая информация.

http://plop.phys.cwru.edu/repairfaq/REPAIR/F_cdfaq.html
Раздел по CD-ROM FAQ’а по ремонту (http://www.repairfaq.org/). Много хорошей технической информации по CD, а также как исправить вещи, когда они ломаются.

http://www.cdmediaworld.com/
Несколько статей по CD-R (a/k/a www.esware.net/empire/hardware).

http://www.cdrwcentral.com/
Новости и информация по записи CD.

Руководство FreeBSD для разработчиков

The FreeBSD Documentation Project

Copyright © 2000, 2001, 2002 by The FreeBSD Documentation Project

Добро пожаловать в руководство FreeBSD для разработчиков. Этот документ находится в просессе написания и представляет собой результат работы множества людей. Многие секции езе не написаны, а некоторые из написаных требуют обновления. Если Вы хотите помочь этому проекту, напишите в Список рассылки Проекта Документации FreeBSD <[email protected]>.

Последняя версия этого документа постоянно доступна с Всемирного Веб Сайта FreeBSD. Этот документ может также быть найден в множестве форматов с FTP Сервера FreeBSD или одного из множества зеркал.

Стандартные инструкции для Lynx

Lynx - программа для работы с WWW в текстовом режиме.

Работая с Lynx, Вы можете в любое время нажать клавишу "?" и получить доступ "Lynx Help Files" (Файлы подсказки для Lynx), составленным разработчиками Lynx. "Lynx help files" более подробны, чем этот документ, но иногда могут представлять большую сложность в использовании.

Инструкции по работе с Lynx

Работая с WWW, пользователь выделяет ссылки в гипертекстовых документах, и, активизируя их, переходит к другому документу. Lynx предоставляет несколько способов для выполнения одного и того же действия. Все они приводят к одному и тому же результату, и Вы можете использовать наиболее удобный для Вас.

Чтобы перейти к документу по выделенной ссылке:

  • Нажмите клавишу "l".
  • Нажмите клавишу Return (Enter).
  • Нажмите "стрелку вправо" ( --> )

Чтобы выделить следующую ссылку на этой же странице

  • Нажмите клавишу "j".
  • Нажмите "стрелку вниз".

Чтобы выделить предыдущую ссылку на этой же странице

  • Нажмите клавишу "k".
  • Нажмите "стрелку вверх".

Чтобы вернуться к предыдущему документу

  • Нажмите клавишу "h".
  • Нажмите "стрелку влево" ( < -- )

Поиск

При работе с Lynx доступны два типа поиска:
  • Поиск слова или фразы в данном документе. Это полезно при работе с большим документом или длинным списком документов. Нажмите клавишу "/", чтобы воспользоваться этим поиском.
  • Второй тип поиска доступен только при работе с некоторыми WWW-серверами. Этот поиск позволяет искать нужное слово или фразу во всех документах сервера. Поскольку этот поиск обеспечивается не программой Lynx, а конкретным WWW-сервером, может оказаться полезным обратиться к файлам подсказки этого сервера или Руководству пользователя (если такое руководство написано ЭКОЛОГИЕЙ). Если этот тип поиска доступен, в нижней части экрана появится строка: "This is a searchable index. Press the "s" key to enter the Search Terms" (Это - индекс, в котором возможен поиск. Нажмите клавишу "s", чтобы ввести условия для поиска).

Сохранение документа на Ваш компьютер

Существуют три способа сохранить документ на Ваш компьютер при помощи Lynx. Вы можете "загрузить" ("download") его, "распечатать" ("print") документ, или послать его по электронной почте ("e-mail") на собственный адрес.

Загрузка

Введите команду "d", чтобы загрузить документ на свой компьютер. Пользователь ГласНета через некоторое время (необходимое для загрузки файла на компьютер ГласНета) увидит следующие опции:
  • "save to disk disabled" - эта опция для пользователей ГласНета недоступна.
  • "Download to local terminal" - Выберите эту опцию. За этим последует обычная процедура загрузки файла на компьютер, с которой Вы могли встречаться при загрузке почты - ГласНет предложит Вам меню для выбора протокола загрузки. Поскольку конкретная процедура загрузки зависит не только от ГласНета, но и от Вашей коммуникационной программы, Вам, возможно, придется проконсультироваться с техническим специалистом.
ВАЖНОЕ ЗАМЕЧАНИЕ: В отличие от методов "распечатки" и "электронной почты", которые сохраняют документ, представленный в данный момент на экране, метод "загрузки" сохраняет документ, на который указывает выделенная ссылка.

Электронная почта

Нажав клавишу "p", вы увидите две опции:

"mail the file to yourself" (послать файл себе по электронной почте)
"print to screen" (распечатать на экран).

Чтобы послать документ по почте:

  1. Нажмите ENTER(RETURN), выделив опцию "mail the file to yourself"
  2. Введите адрес своей электронной почты
  3. Нажмите ENTER(RETURN),
  4. Через некоторое время выгрузите файл из своего обычного почтового ящика

Распечатка

Если Ваша коммуникационная программа поддерживает режим "захвата экрана" (screen capturing):

Нажав клавишу "p", вы увидите две опции:

"mail the file to yourself" (послать файл себе по электронной почте)
"print to screen" (распечатать на экран).

  1. Включите режим "screen capturing" своей коммуникационной программы (Если Вы не знаете, как сделать это, Вам, возможно, придется проконсультироваться с техническим специалистом)
  2. Выберите "print to screen"
  3. После того, как документ будет полностью выдан на экран, отключите режим "screen capturing"
  4. Теперь документ сохранен на Вашем компьютере и может быть распечатан как обычный документ.

Сводка полезных команд:

  • "g" - Перейти по указанному URL (адресу). Эта команда предлагает ввести в специальную строку URL для перехода .
  • "d" - Загрузить документ, ссылка на который выделена. Это позволит Вам загрузить информацию на Ваш компьютер.
  • "p" - Распечатать документ в файл, на экран, послать документ по почте и т.п.
  • "q" - Выход из Lynx (Заглавная Q для быстрого выхода)
  • "/" - Искать заданные слова в текущем документе
  • "s" - Искать документ с заданными словами при помощи индекса (доступно не для всех документов)
  • "z" - Прекратить текущую передачу файла
  • "=" - Показать адрес выделенной ссылки
  • "?" (or "H") - Файлы подсказки к Lynx, созданные авторами программы.
